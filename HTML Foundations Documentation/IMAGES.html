Image Formats & Embing Them

This page covers common image formats.

Common, popular image formats are:

  • .JPG
    Specs of JPG format
    JPG images are designed for handling large color palettes without exorbitantly increasing file size. This makes them great for photos and images. However, on the other hand JPGs don't allow for transperent pixels, which you can see in the white edges of the image below if you look real close:
  • .GIF
    Specs of GIF format
    GIFs are the go-to option for simple animations, but the trade off is that they're somewhat limited in terms of color palette. Never use them for photos.
  • .PNG
    Specs of PNG format
    PNGs are great for anything thats not a photo or animated. For photos, a PNG file of the same quality (as perceived the human eye) would generally be bigger than an equivalent JPG file.This makes them an excellent fit for icons, technical diagrams, logos, etc
  • .SVG
    Specs of SVG format
    SVGs are vector-based graphics format, meaning it can scale up or down to any dimension without loss of quality. . This property makes SVG images a wonderful tool for responsive design. Theyre good for pretty much all the same use cases as PNGs, and you should use them whenever you can.

Image Dimensions

By default, the img element uses the inherit dimensions of its image file. The example JPG, GIF, and PNG images (taken from internetingishard.net) are actually 150 to 150 pixels, while our SVG mochi is only 75 to 75 pixels.

To get our pixel-based images down to the intended size (75 to 75), we can use the img elements width attribute.

For Example: img src='images/mochi.jpg' width='75'

Text Alternatives

Adding alt attributes to your img elements is a best practice. It defines a “text alternative” to the image being displayed. This has an impact on both search engines and users with text-only browsers (e.g., people that use text-to-speech software due to a vision impairment).

For Example: img src='images/mochi.jpg' width='75' alt='A mochi ball in a bubble'
